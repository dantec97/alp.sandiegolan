Downloads & The ALP Download Manager

The ALP Download Manager provides all the games and files needed for clients in one package. Once you have access via Patreon, you’ll be able to use the manager to get everything set up for your LAN party. The manager handles updates, file organization, and ensures everyone has the correct versions.

How it works:
- After joining as a Patreon supporter, you'll receive access instructions.
- The Download Manager will guide you through setup and keep your files up to date.
- No manual code entry or password required—just follow the instructions provided after joining Patreon.

For installation instructions on building the VM in Windows Hyper-V, visit the Servers page.
For help or troubleshooting, check the Calendar and Chat pages.
